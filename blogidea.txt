how to register ios app with firebase

all steps from android blogs

till select app to add

click on ios icon

step 1
fill apple bundle id

how to find
open your project in xcode -> select your app folder from left sidebar

under the targets select your app <  don;t seleetc .test app>

select signing and capabilities tab from tab bar

under the signing section expand ios and you will see bundle identifier for your app
[image]


add optional App nickname (optional)

and app store id and click register[image]


download GoogleService-Info.plist  and click next

now add this file to your project
open your project in xcode
from left sidebar right click on the main folder and in the menu click on  add files to "your project name"
select the downloaded file and 
click on finish
[image]


open root>ios>yourproject>appDelegate.swift

add the following lines

import UIKit
import FirebaseCore


@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

  var window: UIWindow?

  func application(_ application: UIApplication,
    didFinishLaunchingWithOptions launchOptions:
      [UIApplicationLaunchOptionsKey: Any]?) -> Bool {
    FirebaseApp.configure()

    return true
  }
}
