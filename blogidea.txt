how to register android app with firebase - this guide will follow the REACT native ecosystem

open firebase at https://console.firebase.google.com/
select the project you wanna add android app into [see here how to create firebase project]
[image]

open the project overview tab from sidebar and click on add app
it will show platform -> choose android
[image]

under Add Firebase to your Android app
step 1
    Android package name
   you can find your package name for android in 'MainActvoty.kt' file of your peojct

optionally give nick name to ypur app(helpfull if adding multiple android app under same project)App nickname (optional)

[image]

donwload the oogle-services.json file and place it under root>android>app
[image]


in your android project open root>android>build.gradle or build.gradle.kts
[image]


under the dependencies block
add the following line
        classpath("com.google.gms:google-services:4.4.2")
[image]


open root>android>app>build.gradle
at the very top of the file add the folliwng line
apply plugin: "com.google.gms.google-services"
[image]

in the same file under the dependencies block add the following line
   // 🔥 Firebase BoM
    implementation platform('com.google.firebase:firebase-bom:34.7.0')

now back to firebase
click on continue to console


rebuid the app if the app complied without any error congratulations your fiebase is added for android and  you are ready to use any firebase services


Note ->  verify the versions from the firebase, this guide is created by latest versions availabele at the time of publishing
